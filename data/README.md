# Data Sources

Synthetic source-style data for manufacturing marketplace post-order transparency, fulfillment tracking, delay communication, and support resolution.

- `entities.csv`: 36 order journey records.
- `daily_metrics.csv`: 5,040 order journey-day metric rows.
- `source_events.csv`: 760 operational events, QA checks, experiments, and stakeholder asks.
- `recommended_actions.csv`: 220 candidate actions with effort and expected lift.
